# Release Notes - v0.1.0

Release Date: 2026-07-10

This is the initial foundational release of SprintMind AI, setting up the basic development environment and connectivity with database services.

---

## What's New

*   **FastAPI Backend Structure**: Established API gateway setup, lifespan managers, and CORS configurations.
*   **React Frontend Skeleton**: Created Vite SPA skeleton with TypeScript, Tailwind CSS, and global routing.
*   **Supabase Connection**: Configured connection client using the backend service role key and set up frontend offline fallback.
*   **Logging Middleware**: Added request duration logger to track API gateway latencies.
*   **Health Checks**: Created check endpoints (`/` and `/health`) reporting initialization details.
