# Release Notes - v0.2.0

Release Date: 2026-07-11

This release implements a secure, robust user registration, login, and verification system integrated with Supabase Auth and database tables, covered by end-to-end testing.

---

## What's New

*   **FastAPI Auth Service**: Setup service logic and endpoints for `/signup`, `/login`, `/me`, and `/logout`.
*   **Input Schemas Validation**: Added strict Pydantic checks for email format, password size, and workspace roles.
*   **Database user_profiles**: Implemented sync trigger function populated on signup, and applied RLS restrictions targeting authenticated users.
*   **Form Validations**: Upgraded login and signup UI with form validators and custom warning boxes.
*   **Playwright E2E Verification**: Integrated 13 automated browser tests validating authentication scenarios, page responses, loading states, accessibility parameters, and viewport sizes.
